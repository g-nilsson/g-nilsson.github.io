Significance tests

This section will describe the comparisons between sections and their practical and statistical significance.
For all tests, independence and normality is assumed.

ANOVA Results (global significance test)

ANOVA is a difference of means test for multiple groups. It gives a global p-value, which if significant, means that it's likely that a students average score is effected by the students section, in general.

P-value: 0.47

(With significance level = 0.05)
We don't reject the null hypothesis. We don't have statistically significant evidence that any group has a mean different from the others.

F-statistic: 0.967

A larger F-statistic means more difference between groups.
An F-stat of 0.967 means that the variance between sample means is 0.967 times the variance within samples.

Summary statistics (Pairwise significance tests)

This table summarizes the information from pairwise t-tests between each section. The detailed information of each t-test can be found below this table.
The "Count of significant tests" describes how many other sections this specific section is different from, with statistical significance.
If there is a "problem section", this would show up as one section having a clearly larger number in this column.
The final column "Avg effect size among significant" displays the average cohen's d among all significant tests, so the average number of standard deviations the means are in all significant tests.
